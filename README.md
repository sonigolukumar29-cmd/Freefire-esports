# Free Fire Esports Tournament Platform

## Project Description
This project is a comprehensive platform designed for organizing and managing Free Fire esports tournaments. It provides tools for both organizers and players to participate in competitive gaming.

## Features
- **User Registration**: Players can create accounts and manage their profiles.
- **Tournament Creation**: Organizers can set up tournaments with various configurations.
- **Match Scheduling**: Automatic match scheduling and notifications to players.
- **Leaderboards**: Real-time leaderboard to track player rankings.
- **Live Updates**: Updates on match progress and results.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sonigolukumar29-cmd/Freefire-esports.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Freefire-esports
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- After starting the application, access it via [http://localhost:3000](http://localhost:3000).
- Follow the prompts to register or log in to your account.
- Organizers can begin creating tournaments from the dashboard.

## Tech Stack
- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
